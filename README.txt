Foi realizado algumas das pesquisas para a realização deste teste, foi uma das maneiras de descobrir mais o lado da API.
Neste Desafio acabei descobrindo novas funções e novas conexões com o banco de dados.
Foi utilizado uma conexão ao qual conheci em um curso básico de MVC Entity FrameWork, uma forma mais limpa e mais segura de se utilizar.
Nesta aplicação foi utilizada o funcionamento básico de GET,POST,PUT e DELETE, utilizadas como base do seguinte site https://imasters.com.br/dotnet/criando-e-consumindo-uma-web-api-c-parte-01.
As rotas, JSON, conexão com o BD e o Entity FrameWork foram essênciais nesse projeto, para poder fluir.
Foi utilizado também para a visualização do funcionamento do projeto o APP PostMan.
